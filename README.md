##
This is a student's project provided by hexlet. This is very first project of the course. The main purpose is to accustom pupil to JavaScript ecosystem. A result of the project should be a set of "Brain Games" working from console utility.


Project Mentor - [Alexander O](https://github.com/alekgit).
##

## Installation
```sh
$ npm install -g brasid_brain_games
```
[![asciicast](https://asciinema.org/a/211582.svg)](https://asciinema.org/a/211582)
## Игра: "Проверка на четность"

Задача игрока - ответить 'yes', если число четное, и 'no', если число нечетное

### Запуск игры
```sh
$ brain-even
```
[![asciicast](https://asciinema.org/a/211583.svg)](https://asciinema.org/a/211583)


## Игра: "Калькулятор"

Задача игрока - верно вычислить случайное математическое выражение и записать ответ

### Запуск игры
